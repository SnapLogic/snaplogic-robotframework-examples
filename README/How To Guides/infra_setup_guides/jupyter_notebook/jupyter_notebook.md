## Make Commands

| Command               | What it does                      |
| --------------------- | --------------------------------- |
| `make jupyter-start`  | Start JupyterLab                  |
| `make jupyter-stop`   | Stop JupyterLab                   |
| `make jupyter-status` | Show running status               |
| `make jupyter-logs`   | View logs                         |
| `make jupyter-shell`  | Open a shell inside the container |

> Jupyter also starts automatically when you run `make start-services`.
